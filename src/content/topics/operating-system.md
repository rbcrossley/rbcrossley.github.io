An **Operating System (OS)** is system software that acts as an interface between the user and the computer hardware, managing resources and providing services for application software.

## Functions of an OS

- **Process Management** - creates, schedules and terminates processes
- **Memory Management** - allocates and deallocates memory
- **File Management** - creates, deletes, and manages files
- **Device Management** - controls input/output devices
- **Security** - prevents unauthorized access
- **User Interface** - CLI / GUI

## Types of Operating Systems

1. **Batch OS** - processes batches of similar jobs
2. **Time-Sharing OS** - multiple users share system simultaneously
3. **Distributed OS** - multiple computers networked together
4. **Real-Time OS** - strict time constraints
5. **Mobile OS** - Android, iOS

## Popular Operating Systems

- **Windows** - Microsoft
- **macOS** - Apple
- **Linux** - open-source
- **Android / iOS** - mobile
