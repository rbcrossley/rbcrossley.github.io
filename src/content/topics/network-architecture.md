Network architecture refers to the way network devices and services are structured to serve the connectivity needs of client devices.

## Types of Network Architecture

There are two major types of network architecture:

### 1. Peer-to-Peer (P2P) Architecture

In a peer-to-peer network, computers are connected to each other and can share resources without a dedicated server. Each computer acts as both a client and a server.

**Advantages:**
- Easy to set up
- Less expensive
- No need for a dedicated server administrator

### 2. Client-Server Architecture

In this architecture, a central server provides resources and services to client computers. Clients request and servers respond.

**Advantages:**
- Centralized data and security
- Easier to manage
- Better performance for large networks

## Key Components

- **Hardware** - cables, switches, routers, computers
- **Software** - operating systems, network applications
- **Protocols** - rules for data exchange
- **Topology** - physical/logical layout of the network
