A **Data Structure** is a way to organize, manage, and store data so that it can be accessed and modified efficiently.

## Why Data Structures?

- Efficient data management
- Reusability
- Abstraction
- Better algorithm design

## Types of Data Structures

### Linear Data Structures

- **Array** - collection of elements stored at contiguous memory locations
- **Linked List** - linear collection of nodes
- **Stack** - LIFO (Last In, First Out)
- **Queue** - FIFO (First In, First Out)

### Non-Linear Data Structures

- **Tree** - hierarchical structure
- **Graph** - set of nodes connected by edges
- **Hash Table** - key-value pairs

## Operations on Data Structures

- Insertion
- Deletion
- Traversal
- Searching
- Sorting
- Merging
