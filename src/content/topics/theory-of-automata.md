**Theory of Automata** is a branch of theoretical computer science that deals with the design of abstract machines and the computational problems they can solve.

## Key Concepts

- **Alphabet (Σ)** - a finite set of symbols
- **String** - finite sequence of symbols from an alphabet
- **Language** - a set of strings
- **Grammar** - a set of production rules
- **Automaton** - an abstract self-propelled computing device

## Types of Automata

1. **Finite Automata (FA)**
   - Deterministic Finite Automata (DFA)
   - Non-deterministic Finite Automata (NFA)
2. **Pushdown Automata (PDA)**
3. **Linear Bounded Automata (LBA)**
4. **Turing Machine (TM)**

## Applications

- Compiler design
- Lexical analysis
- Pattern matching
- Network protocols
- AI and natural language processing
