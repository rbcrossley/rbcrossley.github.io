A **Database Management System (DBMS)** is software that interacts with end users, applications, and the database itself to capture and analyze data.

## Why DBMS?

- Reduces data redundancy
- Improves data integrity
- Provides data security
- Enables data sharing
- Concurrent access
- Backup and recovery

## Types of DBMS

1. **Relational DBMS (RDBMS)** - MySQL, PostgreSQL, Oracle
2. **NoSQL DBMS** - MongoDB, Cassandra
3. **Object-Oriented DBMS** - db4o
4. **Hierarchical DBMS** - IBM IMS
5. **Network DBMS** - IDMS

## Components of DBMS

- **Hardware** - physical devices
- **Software** - DBMS itself
- **Data** - the actual content
- **Database Access Language** - SQL
- **Procedures** - rules and instructions
- **Database Users** - DBA, application programmer, end user
