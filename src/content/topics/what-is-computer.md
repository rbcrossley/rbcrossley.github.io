A **computer** is an electronic device that manipulates information or data. It has the ability to store, retrieve, and process data.

## Basic Operations

1. **Input** - receives data
2. **Process** - manipulates data
3. **Output** - produces results
4. **Storage** - keeps data for future use

## Characteristics

- **Speed** - performs millions of calculations per second
- **Accuracy** - performs operations without errors
- **Storage** - can store large amounts of data
- **Diligence** - works without getting tired
- **Versatility** - performs different types of tasks
- **Automation** - works without continuous human intervention

## Components

- **Hardware** - physical parts (CPU, RAM, HDD, etc.)
- **Software** - programs and instructions
- **Firmware** - permanent software programmed into read-only memory
