**Computer Organization** refers to the operational units and their interconnections that realize the architectural specifications of a computer.

## Computer Architecture vs Organization

- **Architecture** - what the computer does (instruction set, addressing modes)
- **Organization** - how it does it (control signals, memory technology)

## Basic Functional Units

1. **Input Unit** - keyboard, mouse, scanner
2. **Memory Unit** - primary (RAM) and secondary (HDD, SSD)
3. **Arithmetic Logic Unit (ALU)** - performs arithmetic and logic operations
4. **Control Unit** - directs operations of the processor
5. **Output Unit** - monitor, printer

## Bus System

- **Address Bus** - carries memory addresses
- **Data Bus** - carries data
- **Control Bus** - carries control signals
