The **OSI Model** (Open Systems Interconnection Model) is a conceptual framework used to understand and describe how different networking protocols interact and work together to enable communication between systems.

## The 7 Layers of OSI Model

From top to bottom:

1. **Application Layer** - Closest to the end user; provides network services to applications (HTTP, FTP, SMTP)
2. **Presentation Layer** - Translates data, encryption, compression
3. **Session Layer** - Manages sessions between applications
4. **Transport Layer** - Reliable data transfer (TCP, UDP)
5. **Network Layer** - Routing and addressing (IP)
6. **Data Link Layer** - Node-to-node delivery (MAC)
7. **Physical Layer** - Transmission of raw bits over a medium

## Why OSI Matters

The OSI model serves as a guide for understanding network architecture and designing protocols. It helps in:

- Standardizing communication
- Modular design
- Troubleshooting network issues
- Interoperability between vendors
