# Benefits of network segmentation

Network segmmentation means subnetting.

01) enhance security

Screened subnet can be used to seggregrate web server traffic from internal network traffic.

Size of broadcast domain is decreased.

02) improve performance

If the size of broadcast domain is decreased, it consequently decreases broadcast traffic. This results in better use of bandwidth and better network performance.

03) simplify troubleshooting

Problem can be narrowed down to a smaller network segment.

eg: suppose a network is subdivided dinto:

- Accounting
- HR
- IT

One day there is a trouble transmitting data only to the Accounting network. This fact gives the network administrator some significant insight into the nature of the problem.
